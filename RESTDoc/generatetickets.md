# Generated tickets

A user can print all unsold tickets for certain event.



**URL**: `/api/generatetickets`

**Method**: `POST`

**Auth required**: YES


### Data constraints

```json
{
    "eventId": 1,
    "ticketTypeId": 1
}
```


### Success Response

**Code**: `.`


### Error Response
**Condition**: `.`

**Code**: `.`
**Content**: `.`
